## 🔑 API Configuration & Credentials Setup

The website integrates two serverless components: **Firebase Firestore** and **Groq LLaMA Chat completions**.

### 1. Groq LLaMA 3.3 API Key Setup (`VITE_GROQ_API_KEY`)
The floating chatbot (`src/components/Chatbot.tsx`) requests answers directly from Groq's endpoint.
* **Local Setup**: Stored inside a `.env.local` file at the root of the project:
  ```env
  VITE_GROQ_API_KEY="your_groq_api_key_here"
  ```
  Vite automatically loads keys prefixed with `VITE_` and exposes them in the browser via `import.meta.env.VITE_GROQ_API_KEY`.
* **Production Setup (Vercel)**: Since `.env.local` is gitignored, you must configure this environment variable manually in your Vercel Dashboard under **Settings > Environment Variables** with:
  - **Key**: `VITE_GROQ_API_KEY`
  - **Value**: `your_groq_api_key_here`
